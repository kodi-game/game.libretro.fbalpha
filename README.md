# Arcade (FinalBurn Neo)

### Description

FinalBurn Neo is a fork of FinalBurn Alpha, an emulator of arcade games.

FB Neo emulation includes Capcom CPS-1 and CPS-2, SNK Neo-Geo, Toaplan, Cave and Sega System 16.
